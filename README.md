## Scripts
Use the package manager [yarn](https://yarnpkg.com/)<br />

<i>server</i>

```sh
> npm run start 
```

<i>client</i>

```sh
> yarn run start
> yarn run build
> yarn run test
> yarn run eject
```

<i>uWebSockets</i>

```sh
> yarn run start
> yarn add uNetworking/uWebSockets.js#v18.1.0 --verbose
```
